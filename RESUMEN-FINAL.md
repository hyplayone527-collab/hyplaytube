## 📱 **Para usar la App Móvil:**

1. **Obtener tu IP local:**
   ```bash
   ipconfig
   # Buscar "Dirección IPv4" (ej: 192.168.1.105)
   ```

2. **Actualizar configuración:**
   Editar `mobile-app/src/config/api.js`:
   ```javascript
   BASE_URL: 'http://TU_IP_AQUI:5001/api'
   ```

3. **Ejecutar:**
   ```bash
   cd mobile-app
   npm install
   npm start
   ```
