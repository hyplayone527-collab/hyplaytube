📋 Resumen del Proyecto

NovaStream es una plataforma completa de streaming y redes sociales que combina las mejores características de YouTube, Instagram y TikTok. Desarrollada con tecnologías modernas como React 18, Node.js y Socket.IO.

🎯 Objetivos

  • Crear una experiencia de usuario fluida y moderna
  • Implementar funcionalidades de streaming en tiempo real
  • Desarrollar un sistema de gamificación completo
  • Proporcionar una arquitectura escalable y mantenible

✨ Características Principales

🎥 Sistema de Videos

Subida, reproducción y gestión de contenido multimedia con sistema de likes, comentarios y compartir.

📱 Stories Temporales

Historias que desaparecen en 24 horas, similar a Instagram Stories, con visualización en tiempo real.

💬 Chat en Tiempo Real

Sistema de mensajería instantánea usando WebSocket para comunicación fluida entre usuarios.

🏆 Sistema de Logros

Gamificación completa con achievements, puntos y sistema de recompensas para engagement.

🔔 Notificaciones

Sistema de notificaciones push en tiempo real para mantener a los usuarios conectados.

👥 Red Social

Sistema de seguimiento, perfiles de usuario, feed personalizado y interacciones sociales.

🚀 Instalación y Configuración

Prerrequisitos

Node.js >= 16.0.0
npm >= 8.0.0
Git

Pasos de Instalación

# 1. Clonar el repositorio
git clone https://github.com/hyplayone527-collab/hyplaytube.git
cd hyplaytube

# 2. Instalar dependencias del backend
cd backend
npm install

# 3. Instalar dependencias del frontend
cd ../frontend
npm install

# 4. Configurar variables de entorno
cp .env.example .env
# Editar .env con tus configuraciones

# 5. Ejecutar en modo desarrollo
# Terminal 1 - Backend
cd backend && npm run dev

# Terminal 2 - Frontend
cd frontend && npm run dev

🔌 API Endpoints

Autenticación

POST /api/users/register # Registro de usuario
POST /api/users/login # Inicio de sesión
GET /api/users/profile # Perfil del usuario
PUT /api/users/profile # Actualizar perfil

Posts y Contenido

GET /api/posts # Obtener todos los posts
POST /api/posts # Crear nuevo post
PUT /api/posts/:id/like # Dar like a un post
POST /api/posts/:id/comment # Comentar en un post

Stories

GET /api/stories # Obtener stories activas
POST /api/stories # Crear nueva story
PUT /api/stories/:id/view # Marcar story como vista

🎨 Frontend - React

Tecnologías Utilizadas

React 18 React Router Context API Framer Motion Socket.IO Client Axios

Componentes Principales

  • Navbar: Navegación principal con búsqueda y notificaciones
  • PostCard: Tarjeta de post con likes, comentarios y compartir
  • Stories: Visualizador de historias temporales
  • Chat: Interfaz de mensajería en tiempo real
  • Notifications: Centro de notificaciones

⚙️ Backend - Node.js

Tecnologías Utilizadas

Node.js Express.js Socket.IO JWT Bcrypt JSON Database

Características del Servidor

  • Autenticación JWT: Sistema seguro de tokens
  • Rate Limiting: Protección contra spam y ataques
  • WebSocket: Comunicación en tiempo real
  • Middleware de Seguridad: CORS, Helmet, validaciones
  • Base de Datos JSON: Sistema de almacenamiento simple y eficiente
